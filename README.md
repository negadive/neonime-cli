silahkan download python pada link dibawah

[python 3.7 64bit](https://www.python.org/ftp/python/3.7.9/python-3.7.9-amd64.exe)

[python 3.7 32bit](https://www.python.org/ftp/python/3.7.9/python-3.7.9.exe)
